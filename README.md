# Guild Planner - WoW Discord Bot

Discord bot for World of Warcraft Guilds - Plan your raids more efficiently!

## Preview

### Feature: Planner
![Guild planner - planner feature preview](https://github.com/PatricNox/guild-planner/assets/preview-planner.png "Guild planner - planner feature preview")


## Requirements

* [node](https://nodejs.org/en/) v10 or higher
* [npm](https://www.npmjs.com/) 6.4.x or higher _(usually included with node)_


## Getting Started

1. Fork the repository locally and rename `.env.example` to `.env`
2. Edit the `.env` file for your needs. 
3. Install the bot by running `npm install`
4. Start the bot using `node index.js` in the root folder
